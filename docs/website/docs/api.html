API 文档

概述

Mnemo 提供了完整的命令行界面和 MCP (Model Context Protocol) 接口，支持知识管理、搜索和同步等功能。

知识管理

创建、更新、删除和查询知识条目

语义搜索

基于向量的智能搜索和推荐

MCP 集成

与主流 AI 客户端无缝集成

数据同步

支持多客户端数据同步

CLI 命令

基础命令

mnemo --version

显示版本信息

$ mnemo --version
mnemo 0.1.0

mnemo --help

显示帮助信息

$ mnemo --help
Usage: mnemo [OPTIONS] COMMAND [ARGS]...

Options:
  --version  Show the version and exit.
  --help     Show this message and exit.

Commands:
  search     Search knowledge entries
  create     Create new knowledge entry
  update     Update existing knowledge entry
  delete     Delete knowledge entry
  get        Get knowledge entry by ID or title
  feedback   Record feedback on knowledge entry
  setup      Configure MCP clients
  serve      Start HTTP server
  mcp        Start MCP server

知识管理

mnemo search

搜索知识条目

$ mnemo search [OPTIONS] QUERY

Options:
  --scope TEXT          Scope: global, project, session
  --project-name TEXT   Project name for project scope
  --mode TEXT           Search mode: hybrid, fts, vector
  --sort-by TEXT        Sort: relevance, time, feedback
  --limit INTEGER       Result limit (default: 20)
  --include-archived    Include archived entries

Examples:
  mnemo search "React hooks"
  mnemo search --mode fts "exact phrase"
  mnemo search --scope project "current project decisions"

mnemo create

创建新的知识条目

$ mnemo create [OPTIONS]

Options:
  --title TEXT          Title (required)
  --summary TEXT        Summary (required)
  --content TEXT        Content (required)
  --tags TEXT           Comma-separated tags
  --claim-type TEXT     Type: fact, decision, procedure, hypothesis
  --scope TEXT          Scope: global, project, session
  --project-name TEXT   Project name
  --source TEXT         Source reference
  --related TEXT        Related entries (comma-separated)

Examples:
  mnemo create --title "React Best Practices" \
    --summary "Common React patterns and practices" \
    --content "Detailed explanation..." \
    --tags "react,frontend,web" \
    --claim-type procedure

mnemo get

获取知识条目详情

$ mnemo get [OPTIONS] ID_OR_TITLE

Options:
  --format TEXT   Output format: text, json, markdown

Examples:
  mnemo get 123
  mnemo get "React Best Practices"
  mnemo get --format json 123

mnemo update

更新现有知识条目

$ mnemo update [OPTIONS] ID

Options:
  --title TEXT          New title
  --summary TEXT        New summary
  --content TEXT        New content
  --tags TEXT           New tags
  --claim-type TEXT     New claim type

Examples:
  mnemo update 123 --title "Updated Title"
  mnemo update 123 --content "New content..."

mnemo delete

删除知识条目

$ mnemo delete ID

Examples:
  mnemo delete 123

mnemo feedback

记录知识条目反馈

$ mnemo feedback [OPTIONS] KNOWLEDGE_ID SIGNAL

Options:
  --reason TEXT     Feedback reason
  --actor TEXT      Feedback actor (default: agent:unknown)

Signals:
  helpful      - The entry was useful
  misleading   - The entry was misleading
  outdated     - The entry is outdated

Examples:
  mnemo feedback 123 helpful
  mnemo feedback 123 misleading --reason "Incorrect information"
  mnemo feedback 123 outdated --reason "New API available"

配置和服务器

mnemo setup

配置 MCP 客户端

$ mnemo setup [OPTIONS]

Options:
  --auto                 Auto-configure all detected clients
  --mcp-only            Only configure MCP, don't modify prompts
  --mode TEXT           MCP mode: stdio, http
  --port INTEGER        HTTP port (default: 8787)
  --dry-run             Show what would be done
  --uninstall           Remove configuration
  --no-project-prompts  Skip project-specific prompts

Examples:
  mnemo setup --auto
  mnemo setup --mode http --port 8787
  mnemo setup --dry-run --mcp-only
  mnemo setup --uninstall

mnemo serve

启动 HTTP 服务器

$ mnemo serve [OPTIONS]

Options:
  --host TEXT     Host address (default: localhost)
  --port INTEGER  Port number (default: 8787)
  --cors          Enable CORS

Examples:
  mnemo serve
  mnemo serve --host 0.0.0.0 --port 8080
  mnemo serve --cors

mnemo mcp

启动 MCP 服务器

$ mnemo mcp [OPTIONS]

Options:
  --mode TEXT     MCP mode: stdio, http
  --port INTEGER  HTTP port (default: 8787)

Examples:
  mnemo mcp
  mnemo mcp --mode http --port 8787

MCP API

Mnemo 通过 MCP 协议提供以下工具和资源：

可用工具

mnemo__search

搜索知识条目

参数：
  • query (string, required) - 搜索查询
  • scope (string, optional) - 范围：global/project/session
  • mode (string, optional) - 模式：hybrid/fts/vector
  • limit (integer, optional) - 结果限制

mnemo__create_knowledge

创建知识条目

参数：
  • title (string, required) - 标题
  • summary (string, required) - 摘要
  • content (string, required) - 内容
  • tags (string, required) - 标签（逗号分隔）
  • claim_type (string, optional) - 类型：fact/decision/procedure/hypothesis
  • scope (string, optional) - 范围：global/project/session

mnemo__get_knowledge

获取知识条目

参数：
  • id_or_title (string, required) - ID 或精确标题

mnemo__update_knowledge

更新知识条目

参数：
  • id (integer, required) - 条目 ID
  • title (string, optional) - 新标题
  • summary (string, optional) - 新摘要
  • content (string, optional) - 新内容
  • tags (string, optional) - 新标签

mnemo__feedback_knowledge

记录反馈

参数：
  • knowledge_id (integer, required) - 知识条目 ID
  • signal (string, required) - 信号：helpful/misleading/outdated
  • reason (string, optional) - 反馈原因

HTTP API

当使用 HTTP 模式时，Mnemo 提供 RESTful API：

端点列表

GET /api/v1/search

搜索知识条目

查询参数：
  • q (string, required) - 搜索查询
  • scope (string) - 范围
  • mode (string) - 搜索模式
  • limit (integer) - 结果限制
响应：
{
  "results": [
    {
      "id": 123,
      "title": "React Best Practices",
      "summary": "Common React patterns",
      "tags": ["react", "frontend"],
      "claim_type": "procedure",
      "created_at": "2024-01-01T00:00:00Z",
      "updated_at": "2024-01-01T00:00:00Z"
    }
  ],
  "total": 1,
  "query": "React"
}

POST /api/v1/knowledge

创建知识条目

请求体：
{
  "title": "New Knowledge",
  "summary": "Summary",
  "content": "Full content",
  "tags": "tag1,tag2",
  "claim_type": "fact",
  "scope": "global"
}
响应：
{
  "id": 124,
  "title": "New Knowledge",
  "created_at": "2024-01-01T00:00:00Z"
}

GET /api/v1/knowledge/{id}

获取知识条目

响应：
{
  "id": 123,
  "title": "React Best Practices",
  "summary": "Common React patterns",
  "content": "Detailed content...",
  "tags": ["react", "frontend"],
  "claim_type": "procedure",
  "scope": "global",
  "created_at": "2024-01-01T00:00:00Z",
  "updated_at": "2024-01-01T00:00:00Z"
}

PUT /api/v1/knowledge/{id}

更新知识条目

请求体：
{
  "title": "Updated Title",
  "content": "Updated content"
}

DELETE /api/v1/knowledge/{id}

删除知识条目

响应：
{"message": "Knowledge entry deleted"}

POST /api/v1/knowledge/{id}/feedback

记录反馈

请求体：
{
  "signal": "helpful",
  "reason": "Very useful information"
}

使用示例

CLI 使用示例

基本搜索

# 搜索 React 相关知识
mnemo search "React hooks"

# 在项目范围内搜索
mnemo search --scope project "API design"

# 精确短语搜索
mnemo search --mode fts "exact phrase"

# 语义搜索
mnemo search --mode vector "similar concepts"

创建知识条目

# 创建最佳实践条目
mnemo create \
  --title "React Performance Tips" \
  --summary "Common React performance optimization techniques" \
  --content "Use React.memo, useCallback, useMemo..." \
  --tags "react,performance,optimization" \
  --claim-type procedure

# 创建决策记录
mnemo create \
  --title "Database Choice: PostgreSQL vs MongoDB" \
  --summary "Decision to use PostgreSQL for the main application" \
  --content "After evaluating both options..." \
  --tags "database,postgresql,decision" \
  --claim_type decision

配置 MCP

# 自动配置所有客户端
mnemo setup --auto

# 只配置 MCP，不修改提示词
mnemo setup --mcp-only

# 配置 HTTP 模式
mnemo setup --mode http --port 8787

# 预览将要进行的配置
mnemo setup --dry-run

HTTP API 使用示例

curl 命令

# 搜索知识
curl "http://localhost:8787/api/v1/search?q=React"

# 创建知识
curl -X POST http://localhost:8787/api/v1/knowledge \
  -H "Content-Type: application/json" \
  -d '{
    "title": "New Knowledge",
    "summary": "Summary",
    "content": "Content",
    "tags": "tag1,tag2"
  }'

# 获取知识
curl http://localhost:8787/api/v1/knowledge/123

# 记录反馈
curl -X POST http://localhost:8787/api/v1/knowledge/123/feedback \
  -H "Content-Type: application/json" \
  -d '{"signal": "helpful", "reason": "Very useful"}'

Python 示例

import requests

# 搜索知识
response = requests.get('http://localhost:8787/api/v1/search', params={'q': 'React'})
results = response.json()['results']

# 创建知识
knowledge = {
    'title': 'Python Best Practices',
    'summary': 'Common Python patterns',
    'content': 'Detailed content...',
    'tags': 'python,best-practices'
}
response = requests.post('http://localhost:8787/api/v1/knowledge', json=knowledge)
new_id = response.json()['id']

# 获取知识
response = requests.get(f'http://localhost:8787/api/v1/knowledge/{new_id}')
knowledge_detail = response.json()

错误处理

HTTP 状态码

200 OK

请求成功

400 Bad Request

请求参数错误

404 Not Found

资源不存在

422 Unprocessable Entity

数据验证失败

500 Internal Server Error

服务器内部错误

错误响应格式

{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Invalid input data",
    "details": {
      "field": "title",
      "issue": "Title is required"
    }
  }
}
