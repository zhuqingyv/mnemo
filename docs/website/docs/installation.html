安装指南

快速安装

Mnemo 提供自动化安装脚本，能够检测您的操作系统并安装对应版本。

macOS / Linux

curl -fsSL https://github.com/zhuqingyv/mnemo/releases/latest/download/install.sh | sh

Windows (PowerShell)

irm https://github.com/zhuqingyv/mnemo/releases/latest/download/install.ps1 | iex

安装器功能

  • 自动检测操作系统和架构
  • 下载对应平台的二进制文件
  • 验证 SHA256 校验和
  • 将二进制文件安装到用户目录
  • 自动配置 PATH 环境变量
  • 配置支持的 AI 客户端

手动安装步骤

  1. 下载对应平台的二进制文件
  2. 验证文件完整性（可选但推荐）： 
    # 下载 SHA256SUMS 文件
curl -O https://github.com/zhuqingyv/mnemo/releases/latest/download/SHA256SUMS

# 验证下载的文件
sha256sum -c SHA256SUMS 2>/dev/null | grep OK
  3. 将二进制文件移动到 PATH 目录： 
    # macOS/Linux
mkdir -p ~/.mnemo/bin
mv mnemo-* ~/.mnemo/bin/mnemo
chmod +x ~/.mnemo/bin/mnemo

# Windows
mkdir "%LOCALAPPDATA%\mnemo\bin"
move mnemo-*.exe "%LOCALAPPDATA%\mnemo\bin\mnemo.exe"
  4. 添加到 PATH（如果尚未自动添加）： 
    # 添加到 shell 配置文件
echo 'export PATH="$HOME/.mnemo/bin:$PATH"' >> ~/.zshrc
echo 'export PATH="$HOME/.mnemo/bin:$PATH"' >> ~/.bashrc

# Windows (需要管理员权限)
setx PATH "%PATH%;%LOCALAPPDATA%\mnemo\bin"
  5. 运行自动配置： 
    mnemo setup --auto

安装验证

安装完成后，请验证 Mnemo 是否正确安装：

mnemo --version
mnemo --help

验证步骤

  1. 检查版本信息是否正确显示
  2. 确认帮助信息可以正常显示
  3. 测试搜索功能： 
    mnemo search --help
  4. 验证 MCP 功能： 
    mnemo setup --help | grep -- --mcp-only

配置说明

Mnemo 安装后会自动配置支持的 AI 客户端。以下是配置文件位置：

Claude Code

~/.claude.json

Claude Desktop

~/Library/Application Support/Claude/claude_desktop_config.json

Cursor

~/.cursor/mcp.json

Codex CLI

~/.codex/config.toml

Windsurf

~/.codeium/windsurf/mcp_config.json

GitHub Copilot CLI

~/.copilot/mcp-config.json

MCP 配置格式

Mnemo 默认使用 stdio 模式配置 MCP：

{
  "mcpServers": {
    "mnemo": {
      "command": "/absolute/path/to/mnemo",
      "args": ["mcp"]
    }
  }
}

故障排除

命令未找到

如果运行 mnemo 命令时出现 "command not found" 错误：

  • 确认 ~/.mnemo/bin (POSIX) 或 %LOCALAPPDATA%\mnemo\bin (Windows) 在 PATH 中
  • 重新启动终端或重新登录
  • 手动添加到 PATH（参考手动安装步骤）

权限被拒绝

如果在 macOS 上遇到权限问题：

  • 运行：xattr -d com.apple.quarantine /path/to/mnemo
  • 或者在"系统偏好设置 > 安全性与隐私"中允许运行

MCP 服务器未启动

如果 AI 客户端无法连接到 Mnemo：

  • 重新运行：mnemo setup --auto
  • 重启 AI 客户端
  • 检查配置文件路径是否正确

安装脚本失败

如果自动安装脚本失败：

  • 检查网络连接
  • 确认有足够的磁盘空间
  • 尝试手动安装
  • 查看 GitHub Issues 寻找解决方案

安装后步骤

成功安装 Mnemo 后，您可以：

1. 重启 AI 客户端

重启您的 AI 客户端以加载新的 MCP 服务器配置。

2. 验证连接

在 AI 客户端中询问："你能访问 mnemo 知识库吗？"

3. 开始使用

开始让 AI 客户端使用 Mnemo 记录和检索知识。

4. 查看文档

阅读 API 文档 了解更多功能。
