核心功能

🧠

智能记忆管理

自动记录和检索 agent 的经验、决策和最佳实践，形成可共享的知识网络。

🔄

实时同步

通过 MCP 协议与主流 AI 客户端无缝集成，实现知识的实时同步和共享。

🔍

语义搜索

基于向量的语义搜索技术，快速找到相关的知识和经验。

⚡

高性能

本地 SQLite 数据库，响应迅速，支持大规模知识存储。

🛡️

安全可靠

端到端加密，确保知识安全，支持数据备份和恢复。

🎯

多客户端支持

支持 Claude、Cursor、Codex、Windsurf 等主流 AI 开发工具。

系统架构

Agent 的海马体

Mnemo 采用分布式知识库架构，每个 agent 都可以作为知识的贡献者和消费者。通过 MCP 协议，Mnemo 能够：

  • 自动记录 agent 的决策过程和结果
  • 建立知识之间的关联关系
  • 提供基于上下文的智能推荐
  • 支持知识的版本控制和演进

未来将支持 LoRA 训练，让知识真正内化到模型中。
